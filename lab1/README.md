# Лабораторная работа: Проектирование и настройка корпоративной сети

## **Цель работы**
Спроектировать и настроить адресное пространство для корпоративной сети с филиалами в разных городах, настроить VLAN, управление сетевыми устройствами и оптимизировать использование линков.

### **Домашнее задание**
**Цель:** Настроить корпоративную сеть с филиалами в Москве, Санкт-Петербурге, Лабытнанги и Чокурдахе.

**Задачи:**
1) Разработать и задокументировать адресное пространство для лабораторного стенда
2) Настроить IP адреса на каждом активном порту
3) Настроить каждый VPC в каждом офисе в своем VLAN
4) Настроить VLAN/Loopback интерфейсы управления для сетевых устройств
5) Настроить сети офисов для предотвращения broadcast штормов и оптимизации линков

## **Меры по предотвращению broadcast штормов и оптимизации:**

1. **Сегментация на VLAN:** Разделение клиентов на разные broadcast домены
2. **Настройка STP:** Rapid PVST+ с назначением root bridge для каждого VLAN
3. **Агрегация каналов:** LACP между SW4-SW5 и SW9-SW10
4. **Маршрутизация между VLAN:** Использование SVI на L3 коммутаторах
5. **Storm Control:** Ограничение broadcast трафика на портах доступа
6. **HSRP/VRRP:** Резервирование шлюзов по умолчанию
7. **Route Summarization:** Агрегация маршрутов на граничных маршрутизаторах
8. **ECMP:** Балансировка нагрузки на множественных путях
